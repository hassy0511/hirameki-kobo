# v11 再創造レポート

> この文書は、足し算のみで独立作品化したv11初版の記録である。現在の足し算＋引き算版は `GAME_SPEC.md`、`WORLD_BIBLE_V11.md`、`docs/stage_overview.md` を参照する。

## 結論

v11はv10の改装ではなく、教育設計のみを引き継いだ別作品として再構築した。

## 変更した核

- 作品名を「ひらめき工房と そらのまち」へ変更
- 海賊、船長、宝探し、海図、戦闘を廃止
- 新キャラクター「トト」「モクモ」を新規SVGで制作
- ダジャレ中心の会話を、へんな発明による状況コメディへ変更
- 正解で敵を攻撃する流れを、まちと装置を修理する流れへ変更
- 長い冒険地図を、3区画の修理設計図へ変更
- カード収集を、11部品が一つの再起動装置になる組み立て図鑑へ変更
- ステージ9を、正解の回路をつなぐ独自アクションへ変更
- 保存領域を `lumina_state_v1` とし、旧版の進捗から分離

## 実装範囲

- 初回会話と工房名設定
- ホーム、設計図、問題、ヒントと再回答、結果、図鑑、保護者画面
- 足し算11ステージ、各8問
- 3段階評価、連続正解、パーツ獲得、自己評価
- 端末内保存、音と動きの設定、記録出力、全記録リセット
- PWA、オフラインキャッシュ、SVG/PNGアプリアイコン

## 検証

- JavaScript構文確認
- manifest JSON確認
- 新規SVGから192px/512pxアイコンを生成
- 11ステージで3,520問を生成し、正解範囲・選択肢重複・正解包含・ヒント・解説を自動検査
- ステージ1から11まで満点で連続完了し、全11パーツ・全33印の獲得を自動検査

自動検査は `tests/smoke-v11.mjs` で再実行できる。
