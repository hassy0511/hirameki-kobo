# カリキュラム拡張フレームワーク

## 基本単位

ひとつの単元を、ひとつの空島として扱う。空島の中に、導入、練習、中間確認、応用、総合確認を持つ複数ステージを置く。

## 共通の進み方

1. 具体物を操作する
2. 視覚的な補助を使って考える
3. 記号や文字だけで解く
4. 形式を混ぜて理解を確認する
5. 単元固有のアクションで反復する
6. 日常場面や文章へ応用する

## 作品との分離

学習目標と難易度はカリキュラムファイルで管理し、キャラクター、会話、背景、報酬、演出とは分ける。世界観を変えても学習順序が壊れず、学習順序を調整しても物語全体を書き換えずに済む構成を目指す。

新しい空島は、既存ステージの色替えではなく、その単元に合う操作と「直したくなる装置」を一つ以上持たせる。
