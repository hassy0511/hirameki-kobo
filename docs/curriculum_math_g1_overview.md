# 小学1年生・算数カリキュラム全体案

## 位置付け

現行の令和6年度版小学校算数教科書について、東京書籍、大日本図書、日本文教出版の単元と指導内容を比較し、ひらめき工房用の共通技能へ整理した全体案である。

教科書会社ごとの単元名や掲載順をそのまま採用せず、3社に共通する学習内容を `canonicalSkillId` 単位に統合する。学習指導要領の領域A〜Dをすべて含める。

## 6本の学習ライン

| ライン | 主な内容 | 状態 |
|---|---|---|
| かず 基礎ライン | 個数、数字、順序、大小、20・100までの数 | 実装済み |
| たしざん 組み立てライン | 合併、増加、10まで、20まで、繰り上がり | 実装済み |
| ひきざん 整理ライン | 求残、求補、求差、繰り下がり | 実装済み |
| くらべる 計測ライン | 長さ、かさ、広さ、時刻 | 実装済み |
| かたち 設計ライン | 立体、平面、形の構成、位置 | 実装済み |
| しらべる 解決ライン | 分類、絵グラフ、演算決定、文章題、等分 | 実装済み |

## 推奨する年間順序

ラインは自由に選べるが、工房コントロール室では次を「おすすめ」として表示する。

- 1学期前半：かず1〜8 → たしざん1〜5 → ひきざん1〜5
- 1学期後半：しらべる1〜5 → 計測1〜5
- 2学期前半：かず9 → たしざん6〜9 → 計測6〜9
- 2学期後半：たしざん10 → かたち1〜9 → ひきざん6〜9
- 3学期：かず10〜11 → ひきざん10 → 計測10 → 各ライン11 → しらべる6〜11

学校や教科書によって単元順が異なるため、未習単元を一律にロックしない。おすすめ表示とライン内の段階解放を分けて管理する。

## 実装ルール

- 1ステージ8問。操作問題を中心にし、短い場面・文章問題と確認問題を混ぜる
- ステージ5は直前4技能、ステージ11はライン全体の確認
- 数値、対象物、配置、左右、文章場面、選択肢順をseed付きで変える
- 同一プレイ内のsemantic signatureを重複させず、直近32問も可能な限り避ける
- 全11ステージ修理後、そのラインの12問制タイムアタックを解放する
- タイムアタックはミス1回につき3秒加算し、ライン別の自己ベストを保存する
