# 二つの空島 — 全22ステージ一覧

ホームで島を切り替え、それぞれ11個の装置を直す。両方の第1ステージは最初から遊べ、島ごとに進捗・ひらめき印・修理パーツを記録する。

## ひきざんの かげじま

## 共通ルール

- 1ステージ8問、最初の回答で5問以上正解すると修理完了
- 5〜6問で印1個、7問で2個、8問で3個
- 不正解後は、具体的な見方のヒントを表示して同じ問題へ再挑戦
- ステージ5は中間確認、ステージ9はアクションラウンド、ステージ11は総合修理
- 一方の島の進捗は、もう一方の島のアンロック条件にしない
