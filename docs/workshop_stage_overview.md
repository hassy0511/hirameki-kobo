## 共通ルール

- 1ステージ8問、最初の回答で5問以上正解すると修理完了
- 5〜6問で印1個、7問で2個、8問で3個
- 不正解後は具体的な見方のヒントを表示して同じ問題へ再挑戦
- 問題はseed付きで生成し、同一プレイと直近プレイの重複を避ける
- 一方のラインの進捗を、別ラインのアンロック条件にしない
- 各ライン11ステージ完了後、12問制タイムアタックを解放
- タイムアタックのミスは1回3秒加算し、自己ベストをライン別保存
- 既存の足し算・引き算ステージIDは保存データ互換のため維持する
