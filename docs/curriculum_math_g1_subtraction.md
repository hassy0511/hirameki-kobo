# 小学1年生・引き算カリキュラム

共有された修正版教材PDF全49ページを目視確認し、画像や固有文章を転載せず、学習順序と問題構造だけを整理した実装基準である。

## 到達目標

- 5・7・10を2つの数に分け、引き算の土台となる数の組を理解できる
- 具体物を取り除いて、10までの引き算を式と結びつけられる
- 0を引く計算と、同じ数を引く計算を理解できる
- 20までの数を10と端数に分け、繰り下がりのない引き算を解ける
- 3つの数の加減を左から順に計算できる
- 10を使う2つの方法で、繰り下がりのある引き算を解ける
- 100までの位取りと、借りのない「何十−何十」「2桁−1桁」を解ける
- 短い文章から引き算を選び、残りや違いを求められる

## 生成条件

- 10まで：`0 ≤ 引く数 ≤ はじめの数 ≤ 10`
- 0と同数は、導入ステージで意図的に出題する
- 20まで・繰り下がりなし：11〜20から1桁を引き、答えを10以上に保つ
- 3項引き算：途中結果と最終結果を0以上に保ち、必ず左から順に扱う
- 3項加減混合：各途中結果を0〜20に保つ
- 繰り下がり：11〜18から、1の位より大きい1桁を引き、答えを1〜9にする
- 100まで：何十−何十、または2桁−1桁で1の位から引ける問題だけにする
- 教材範囲にない2桁−2桁の繰り下がりは生成しない

## 繰り下がりの2方式

### 10と残りに分ける

例：`13 − 9`

1. 13を10と3に分ける
2. 10−9＝1
3. 1＋3＝4

### 引く数を分けて10にする

例：`13 − 4`

1. 4を3と1に分ける
2. 13−3＝10
3. 10−1＝9

ステージ8の前半は図を常に表示し、後半で補助を減らす。不正解時は「数え直す」だけでなく、どの数を分けるかを具体的に示す。

## 問題形式

- 数の組と空欄
- 取り除いた物を薄く表示する具体物問題
- 数字入力と選択問題
- 残りを求める文章題
- 2つの量の違いを求める問題
- 数直線を後ろへ進む問題
- 3項の途中結果
- 10のまとまりをほどく図
- 十の位と一の位を分けた図
- 正しい着地点を選ぶアクション

## 評価

足し算と共通で1ステージ8問。最初の回答で5問以上正解すると完了し、5〜6問で印1個、7問で2個、8問で3個とする。再回答は学びとして記録するが、最初の正解数には加えない。
