# 小学1年生・足し算カリキュラム

このファイルは物語や画面デザインから独立した、学習順序の基準である。

## 到達目標

- 具体物を数えて、10までの足し算を理解できる
- 5・7・10を2つの数に分けたり、2つの数から作ったりできる
- 20までの数と、繰り上がりのない足し算を扱える
- 3つの数の足し算を扱える
- 10を作る考え方で、繰り上がりのある足し算を解ける
- 短い文章から足し算を選び、式と答えを出せる

## 11段階

1. 10までの数を数える
2. 5・7・10の数の合成と分解
3. 絵で理解する、答えが10までの足し算
4. 数式で解く、答えが10までの足し算
5. 1〜4の混合チェック
6. 10のまとまりを使って20まで数える
7. 繰り上がりのない、答えが20までの足し算
8. 3つの数の足し算
9. 既習内容をテンポよく反復するアクションラウンド
10. 10を作る、繰り上がりのある足し算
11. 数・式・文章題を混ぜた総合チェック

## 足場の外し方

具体物の絵 → まとまりを示す枠 → 数式と絵 → 数式だけ → 穴埋め → 短い文章題、の順で補助を減らす。

不正解時は正解だけを見せず、「何を数えるか」「あといくつで10か」「どの2つを先にまとめるか」のいずれかを具体的に示す。

## 判定

- 1ステージ8問
- 最初の回答で5問以上正解すると完了
- 5〜6問で印1個、7問で2個、8問で3個
- 不正解後はヒントを見て同じ問題へ再挑戦できる
- 再回答の正解は学びとして記録するが、ステージ得点には加えない

## 問題品質の条件

- 必ず一意の正解がある
- 数の範囲が学習段階を超えない
- 選択肢は重複せず、必ず正解を含む
- 文章題は低学年が一度で読める長さにする
- 笑いの文は計算に不要な情報と明確に分ける
