# ひらめき工房と そらのまち

小学1年生の足し算と引き算を、二つの空島の修理として楽しむブラウザゲームです。

公開版：<https://hassy0511.github.io/hirameki-kobo/>

## 遊び方

公開版をブラウザで開くか、`index.html` をローカルのブラウザで開くと遊べます。最初に工房の名前を決め、「たしざんの そらじま」と「ひきざんの かげじま」にある装置を直します。

- 各島11ステージ、合計22ステージ
- 1ステージ8問
- 最初の回答で5問以上正解すると修理完了
- 不正解後は具体的なヒントを見て再挑戦
- 島ごとに11個の修理パーツを集めると、大きな装置が完成

記録はブラウザ内だけに保存されます。通信やアカウント登録はありません。

## iPadでアプリのように遊ぶ

公開版をiPadのSafariで開き、ゲーム内の「このiPadに ついか」から案内を表示します。Safariの共有ボタンを押し、「ホーム画面に追加」→「追加」の順に選ぶと、ホーム画面のアイコンから全画面で起動できます。「Webアプリとして開く」が表示された場合はオンにします。

ホーム画面へ追加した後も、学習記録はそのiPad内に保存されます。更新があるときは、ゲーム内に表示される更新ボタンから新しい版へ切り替えられます。

## 作品の方針

旧作の教育設計だけを受け継いだ独立作品です。世界観、キャラクター、会話、画像、ゲームの見せ方は新規に制作しています。詳細は [WORLD_BIBLE_V11.md](WORLD_BIBLE_V11.md) を参照してください。

## 主なファイル

- `index.html` — ゲーム本体
- `WORLD_BIBLE_V11.md` — 世界観と体験設計
- `docs/curriculum_math_g1_addition.md` — 足し算の学習設計
- `docs/curriculum_math_g1_subtraction.md` — 引き算の学習設計
- `docs/stage_overview.md` — 二つの島・全22ステージ一覧
- `tests/smoke-v11.mjs` — 問題生成と全クリア動線の自動検査
- `manifest.json` / `sw.js` — インストール・オフライン対応

## 自動検査

Node.js がある環境で次を実行します。

```text
node tests/smoke-v11.mjs
```
