# ひらめき工房

小学1・2年生の算数を、工房の装置をさわって動かしながら学ぶブラウザゲームです。現行学習指導要領と令和6年度版教科書複数社の共通内容を、学年ごとに6本の学習ライン・66ステージ、合計132ステージへ整理しています。

公開版：<https://hassy0511.github.io/hirameki-kobo/>

## 小学1年生

- かず 基礎ライン
- たしざん 組み立てライン
- ひきざん 整理ライン
- くらべる 計測ライン
- かたち 設計ライン
- しらべる 解決ライン

各ラインは11ステージ、1ステージは8問です。分類、点灯、数枠、数直線、位取り、時計、計測、図形盤、グラフ、関係図などの操作を中心にし、文章・場面問題と確認テストを組み込んでいます。

小学1年生では、工房中央の中枢装置「ルミナ」を直す物語を全66ステージへ統合しています。プレイヤーは工房の親方となり、6ラインそれぞれの故障を直して復旧コアを作ります。ステージを初めて直すたびに工房固有の変化が起き、11ステージでライン完成、66ステージで「はじまり区画」の結末へ進みます。

問題は毎回seed付きで生成します。同一プレイ内の重複を禁止し、ステージごとの直近出題も保存して再挑戦時の重複を避けます。

## タイムアタック

一つのラインの全11ステージを修理すると、そのラインの「ひらめきタイムアタック」が解放されます。

- 毎回内容が変わる12ミッション
- ミス1回につき3秒加算
- 操作タイム、公式タイム、ミス数を表示
- ライン別の自己ベストと挑戦回数を端末へ保存

## 遊び方

公開版をブラウザで開くか、`index.html` をローカルのブラウザで開きます。記録はブラウザ内だけに保存され、学習中の通信やアカウント登録はありません。

iPadではSafariの共有ボタンから「ホーム画面に追加」を選ぶと、全画面のPWAとして利用できます。

## 小学2年生

文部科学省の現行学習指導要領と、東京書籍・大日本図書・日本文教出版の令和6年度版教科書資料を集約し、小学2年生も6ライン・全66ステージへ整理しました。

- 大きなかず 位取りライン
- たしひき 筆算ライン
- かけ算 増幅ライン
- はかる 計測ライン
- かたち 設計ライン2
- 調査・問題解決ライン

学習目標、中心操作、前提技能、確認テスト、12問制タイムアタック配分は `grade2-curriculum.js` に定義しています。コース選択から二年生を直接開始でき、全66ステージ、文章・場面問題、確認テスト、全6ラインのタイムアタックを遊べます。一年生と二年生の記録はstate v4内で完全に分離します。

## 小学3年生の準備状況

小学3年生は、文部科学省と同じ教科書3社を集約し、内容量に合わせて7ライン・全77ステージへ整理しました。

- 大きな数・たしひき 制御ライン
- かけ算 増幅ライン3
- わり算 分配ライン
- 小数・分数 精密ライン
- はかる 計測ライン3
- かたち 設計ライン3
- 調査・問題解決ライン3

学習目標、中心操作、前提技能、四年生へ送る境界、確認テスト、12問制タイムアタック配分は `grade3-curriculum.js` に定義済みです。小学3年生も公開版へは未接続で、共通の学年別保存基盤と実問題ビルダーを実装してから表示します。

## 作品の方針

舞台は一つの「ひらめき工房」です。単元は工房内の学習ラインとして扱います。ただ計算式を解くだけでなく、考えた内容が装置の動きとして見える体験を重視します。

詳細は [WORKSHOP_BIBLE.md](WORKSHOP_BIBLE.md)、[GAME_SPEC.md](GAME_SPEC.md)、[docs/story_world_and_g1.md](docs/story_world_and_g1.md) を参照してください。

## 自動検査

Node.jsがある環境では次を実行します。

```text
node tests/smoke-v13.mjs
node tests/smoke-v14.mjs
node tests/story-g1-smoke.mjs
node tests/grade2-curriculum-smoke.mjs
node tests/grade3-curriculum-smoke.mjs
```

小学1年生は6,336問の大量生成と全66ステージ回帰を固定しています。統合検査では1・2年生全132ステージ、二年生2,112問、全66ステージ完走、6ラインのタイムアタック、学年間の記録分離、v1〜v3からstate v4への移行、旧データの一度限りのバックアップ、将来version保護、コース選択、PWA資産を確認します。

物語検査では、全体導入の改訂、6ライン固有の物語、66件すべて異なる工房変化、初回だけの導入、ライン完成、高速点検、一年生完走、既存進捗保持、学年間分離、HTML安全性を確認します。

二年生の単体検査では、6ライン・66ステージ、ID一意性、学習指導要領対応、操作メタデータ、確認ステージ、12枠のタイムアタック配分を確認します。

三年生の単体検査では、7ライン・77ステージに加え、教科書3社の時数、一億・限定除法・小数第一位・簡単な同分母分数・角・データの学年境界を確認します。
