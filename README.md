# ひらめき工房

小学1年生の算数を、工房の装置をさわって動かしながら学ぶブラウザゲームです。現行学習指導要領と令和6年度版教科書複数社の共通内容を、6本の学習ライン・全66ステージへ整理しています。

公開版：<https://hassy0511.github.io/hirameki-kobo/>

## 内容

- かず 基礎ライン
- たしざん 組み立てライン
- ひきざん 整理ライン
- くらべる 計測ライン
- かたち 設計ライン
- しらべる 解決ライン

各ラインは11ステージ、1ステージは8問です。分類、点灯、数枠、数直線、位取り、時計、計測、図形盤、グラフ、関係図などの操作を中心にし、文章・場面問題と確認テストを組み込んでいます。

問題は毎回seed付きで生成します。同一プレイ内の重複を禁止し、ステージごとの直近出題も保存して再挑戦時の重複を避けます。

## タイムアタック

一つのラインの全11ステージを修理すると、そのラインの「ひらめきタイムアタック」が解放されます。

- 毎回内容が変わる12ミッション
- ミス1回につき3秒加算
- 操作タイム、公式タイム、ミス数を表示
- ライン別の自己ベストと挑戦回数を端末へ保存

## 遊び方

公開版をブラウザで開くか、`index.html` をローカルのブラウザで開きます。記録はブラウザ内だけに保存され、学習中の通信やアカウント登録はありません。

iPadではSafariの共有ボタンから「ホーム画面に追加」を選ぶと、全画面のPWAとして利用できます。

## 小学2年生の準備状況

文部科学省の現行学習指導要領と、東京書籍・大日本図書・日本文教出版の令和6年度版教科書資料を集約し、小学2年生も6ライン・全66ステージへ整理しました。

- 大きなかず 位取りライン
- たしひき 筆算ライン
- かけ算 増幅ライン
- はかる 計測ライン
- かたち 設計ライン2
- 調査・問題解決ライン

学習目標、中心操作、前提技能、確認テスト、12問制タイムアタック配分は `grade2-curriculum.js` に定義済みです。現在の公開版はまだ小学1年生のみで、小学2年生は学年別の保存形式と問題ビルダーを実装してから表示します。

## 作品の方針

舞台は一つの「ひらめき工房」です。単元は工房内の学習ラインとして扱います。ただ計算式を解くだけでなく、考えた内容が装置の動きとして見える体験を重視します。

詳細は [WORKSHOP_BIBLE.md](WORKSHOP_BIBLE.md) と [GAME_SPEC.md](GAME_SPEC.md) を参照してください。

## 主なファイル

- `index.html` — アプリの入口
- `styles.css` — 画面と操作装置
- `game-core.js` — 全66ステージ、ランダム問題生成、データ移行
- `app.js` — 画面、回答、評価、タイムアタック、保存
- `WORKSHOP_BIBLE.md` — 工房の世界観と体験設計
- `GAME_SPEC.md` — 製品・画面・問題・データ仕様
- `docs/curriculum_math_g1_overview.md` — 小学1年生算数の全体構成
- `docs/curriculum_math_g2_overview.md` — 小学2年生算数の教科書集約と全66ステージ
- `docs/implementation_math_g2.md` — 学年別保存と二年生追加の安全な実装順
- `grade2-curriculum.js` — 小学2年生の実装用ステージメタデータ
- `docs/workshop_stage_overview.md` — 稼働中の全66ステージ
- `tests/smoke-v13.mjs` — 全生成・全完走・移行・タイムアタック検査
- `tests/grade2-curriculum-smoke.mjs` — 二年生66ステージの定義検査
- `manifest.json` / `sw.js` — インストール・オフライン対応

## 自動検査

Node.jsがある環境では次を実行します。

```text
node tests/smoke-v13.mjs
node tests/grade2-curriculum-smoke.mjs
```

検査内容は、6ライン・66ステージ、6,336問の大量生成、直近問題の重複回避、全ステージ完走、タイムアタック、旧保存データ移行、PWA要件です。

二年生の単体検査では、6ライン・66ステージ、ID一意性、学習指導要領対応、操作メタデータ、確認ステージ、12枠のタイムアタック配分を確認します。
